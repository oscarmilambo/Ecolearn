## üìã How to Create a Challenge

### Step 1: Access Challenge Management
1. Log in to Admin Dashboard
2. Click "Community Challenges" in sidebar
3. Click "Create New Challenge" button

### Step 2: Fill in Challenge Details

**Required Fields:**
- **Title:** Catchy, motivating name (e.g., "Clean Kalingalinga Campaign")
- **Description:** Clear instructions and expectations
- **Challenge Type:** Select from dropdown (cleanup, recycling, education, reporting)
- **Start Date & Time:** When challenge begins
- **End Date & Time:** When challenge ends
- **Target Goal:** Number to achieve (e.g., 500 bags, 100 reports)
- **Reward Points:** Points awarded upon completion

**Optional Fields:**
- **Challenge Image:** Inspiring photo to motivate participants

### Step 3: Review and Create
- Double-check all details
- Click "Create Challenge"
- Challenge is now live!

## ‚öôÔ∏è Managing Challenges

### Edit Challenge
1. Go to challenge detail page
2. Update title, description, goals, or rewards
3. Upload new image if needed
4. Click "Update Challenge"

### Toggle Status
- **Activate:** Make challenge active and visible
- **Cancel:** Deactivate challenge (participants can't contribute)

### Award Points
1. Wait until challenge ends or goal is reached
2. Click "Award Points to All"
3. All participants receive reward points automatically
4. Users get notification of reward

### Delete Challenge
1. Go to challenge detail page
2. Click "Delete Challenge"
3. Confirm deletion
4. Challenge and all data removed

## üéÅ Reward System

### Setting Rewards
- **Low Effort:** 50-100 points (e.g., complete 1 module)
- **Medium Effort:** 100-250 points (e.g., collect 50 bottles)
- **High Effort:** 250-500 points (e.g., organize cleanup event)
- **Major Achievement:** 500+ points (e.g., lead community campaign)

### Awarding Points
- Points awarded when challenge completes
- Admin clicks "Award Points to All"
- System creates point transaction for each participant
- Users notified of reward

## üìà Best Practices

### Creating Effective Challenges

1. **Set Realistic Goals**
   - Consider community size
   - Account for available time
   - Make it achievable but challenging

2. **Clear Instructions**
   - Explain what participants need to do
   - Define success criteria
   - Provide examples

3. **Attractive Rewards**
   - Offer meaningful points
   - Consider badges or recognition
   - Celebrate top contributors

4. **Appropriate Duration**
   - Short challenges: 1-2 weeks (quick wins)
   - Medium challenges: 1 month (sustained effort)
   - Long challenges: 2-3 months (major campaigns)

5. **Engaging Titles**
   - Use action words
   - Include location if relevant
   - Make it exciting

### Examples of Good Challenges

‚úÖ **Good:** "Clean Kalingalinga - Collect 500 Bags in 2 Weeks"
- Clear goal
- Specific location
- Defined timeframe

‚ùå **Bad:** "Cleanup Challenge"
- Too vague
- No specific goal
- No location or timeframe

‚úÖ **Good:** "Recycling Champion - Collect 100 Plastic Bottles This Month"
- Specific target
- Clear timeframe
- Motivating title

‚ùå **Bad:** "Recycle Stuff"
- No target
- No timeframe
- Uninspiring

## üéØ Challenge Status Explained

### Upcoming
- Start date is in the future
- Users can see but not participate yet
- Badge: Blue "Upcoming"

### Active
- Currently running
- Between start and end dates
- Users can participate
- Badge: Green "Active"

### Completed
- End date has passed
- Goal may or may not be reached
- No new contributions accepted
- Badge: Gray "Completed"

### Cancelled
- Admin deactivated challenge
- No longer visible to users
- Participants can't contribute
- Badge: Red "Cancelled"
