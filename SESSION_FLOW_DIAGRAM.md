## Summary

### Key Points:

1. **Active users never timeout** (session resets on every page)
2. **Inactive users timeout after 1 hour** (no activity)
3. **Expired sessions redirect to login** then back to intended page
4. **Protected pages require login** (all authenticated features)
5. **Public pages work without login** (landing, about, features, contact)
6. **Browser close destroys session** (must login again)
7. **Logout redirects to landing page** (clean exit)

### Settings:
```python
SESSION_COOKIE_AGE = 3600              # 1 hour
SESSION_SAVE_EVERY_REQUEST = True      # Auto-refresh âœ…
SESSION_EXPIRE_AT_BROWSER_CLOSE = True # Destroy on close
LOGIN_URL = '/accounts/login/'         # Where to redirect
LOGOUT_REDIRECT_URL = '/'              # Landing page
```
